Flag Beschreibung
-o [Dateiname] definiert die Outputdatei
-Ox legt ein Optimierungslevel fest, x = 0 resultiert in keiner Optimierung, x = 3 erwirkt eine maximale Optimierung
-march=[Plattform] bestimmt die Zielplattform, für diese Platine lautet jene "ARM1176JZF-s"
-mtune=[Architektur] bestimmt die Prozessorarchitektur, in diesem Fall heißt jene "ARMv6-ZK"
-mfpu=[Hardware] definiert die Floating-Point-Hardware, hier wird der Wert "vfp" verwendet
-mfloat-abi=[Art] legt den Umgang mit Floating-Point-Variablen fest, "hard" verwendet die Floating-Point-Hardware für Berechnungen
-W[Warnung] unterdrückt die angegebene Warnungsart
-nostartfiles verhindert das Linken der Standard-Startfiles, sodass nur eigener Code ausgeführt wird
